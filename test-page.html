How to Test the Extension:

  1. Click the extension icon in your Chrome toolbar
  2. Click "Activate Overlay Tool" to start
  3. Upload a design image using the floating panel
  4. Try dragging, rotating, and adjusting the overlay
  5. Test keyboard shortcuts (arrow keys, delete)
  6. Verify the panel can be minimized and moved
